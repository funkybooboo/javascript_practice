## run

run `nodemon index.js` to start app in dev mode
run `node index.js` to start app in depoly mode

## environment vaibles

example use `export DEBUG=app:startup`

- DEBUG=
    - app:startup
    - app:database
    - app:*
    - app:startup,app:database
- NODE_ENV=
    - development
        - default
    - production
- app_mail_password=
    - whatever the password is for the server
